Fantasy Battle Sprites
======================

These sprites were generated for Cloud Chess battle captures with the built-in
image generation workflow, then cut out from a chroma-key source sheet.

Prompt summary:

- Cohesive 4 by 2 fantasy chess battle sprite sheet
- Flat #00ff00 chroma-key background
- Pawn with sword and shield
- Archer with bow
- Knight as armored horse-headed lancer
- Bishop as mystical spellcaster
- Rook as living stone tower warrior
- Queen as battle mage
- King as armored monarch
- Dragon with red wings and fire breath

The browser game loads the transparent PNG cutouts directly. The source sheet is
kept for future recropping or style iteration.
